# QuickPick

QuickPick is a fast, explainable shortlist assistant for consumer product choices.

## Run locally

```
npm install
npm run dev
```

Open http://localhost:3000.

## Env

Create a `.env` file:

```
AI_BUILDER_TOKEN=your_token
AI_BUILDER_API_BASE=https://space.ai-builders.com/backend
AI_BUILDER_MODEL=gemini-3-flash-preview
```
